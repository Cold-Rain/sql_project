# Анализ рынка вакансий: итоговый проект по SQL и базам данных


## О проекте

Проект представляет собой комплексный анализ рынка труда на основе реального датасета вакансий. С использованием SQL-запросов к базе данных **CloudRagDB** (MS SQL Server 2025) проведена детальная разведка данных, выявлены ключевые закономерности рынка и сформированы практические выводы для соискателей, рекрутеров и аналитиков HR-сферы.

**Объём данных:** 49 197 вакансий от 23 501 работодателя из 1 362 регионов, охватывающих 294 отрасли экономики.

---

## Цель исследования

Выявить структурные особенности рынка труда через призму:
- Географической концентрации вакансий
- Требований к квалификации и опыту
- Зарплатной прозрачности и уровней компенсации
- Специфики востребованных профессий (на примере Data Scientist)
- Поведения работодателей (география, масштаб, отраслевая принадлежность)
